# Selenium JS – Automation Demo

This repo contains Selenium automation tests in JavaScript:

- **Login Test** – Login form automation  
- **Hover Test** – Mouse hover effect on avatars  
- **Keyboard Test** – Login using keyboard actions  
- **Alert Test** – JavaScript alert handling  
- **iFrame Test** – Interact with elements inside an iFrame  

---

## **Setup**

1. Install Node.js and Chrome  
2. Install Selenium WebDriver:

```bash
npm install selenium-webdriver
